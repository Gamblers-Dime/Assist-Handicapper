SECTION 1 — AGENT OVERVIEW & PURPOSE
1.1  What This Agent Does
The Basketball-IQ CourtIQ AI Agent is an autonomous NBA prop-betting intelligence system. It continuously monitors online sportsbook lines, applies the CourtIQ v3.0 predictive algorithm to every available PG assists, turnovers, and team assists over/under proposition, and automatically places bets when the model's win probability exceeds defined thresholds.
The agent operates across a defined daily cycle: data ingestion → line retrieval → algorithmic scoring → bet decision → order placement → result tracking → model feedback.
1.2  Core Responsibilities
The agent must perform the following functions autonomously:
* Monitor DraftKings and partner books for all available NBA PG prop lines (assists, turnovers, PRA, team assists)
* Ingest real-time injury signals from Sleeper's public API and flag affected PGs
* Retrieve NBA player game logs, team defense ratings, and coach pace metrics from stats.nba.com
* Run the 13-factor Bayesian regression model on each available line
* Execute Monte Carlo simulation (10,000 samples) to establish 90th-percentile confidence range
* Classify each bet as Grade A (≥76% conf, ≥6% edge), Grade B (≥70% conf, ≥4% edge), or Skip
* Apply win probability thresholds to determine stake amount ($5 / $30 / $100)
* Place bets via sportsbook API connections with appropriate stake and bet type
* Log all decisions, outcomes, and model performance for continuous improvement
* Answer user queries about NBA players, teams, coaches, and matchup context

	Steps 5-7 — Confidence Scoring, Monte Carlo, and Kelly:
After computing the model projection, the algorithm:
1. Calculates win probability using a logistic function: P(win) = 1 / (1 + e^(−1.7 × z)) where z = (projection − line) / std_dev
2. Runs 10,000 Monte Carlo simulations using Box-Muller normal distribution sampling
3. Requires the 90th-percentile MC range to clear the line before issuing a bet signal
4. Applies ¼ Kelly Criterion: Kelly% = (b×p − q) / b, capped at 15% of bankroll
5. Assigns Grade: A (conf ≥ 76% AND edge ≥ 6%), B (conf ≥ 70% AND edge ≥ 4%), or Skip

	5.2  Model Uncertainty Disclosures
The agent must communicate uncertainty to users in all of the following situations:
* When win probability is between 70–72%: "This is a borderline signal — slightly above threshold. Treating as $5 minimum bet."
* When flagWarnings contains any entry: "Model flags [warning]. Adjusting confidence downward. Proceeding at reduced stake."
* When a PG has played fewer than 20 games this season: "Limited sample size — season average less reliable. Weighting L10 more heavily."
* When the line has moved more than 0.5 units from open: "Line has moved significantly. This may indicate sharp money or injury news. Re-running model with current line before betting."
* When backtested win rate for this bet type is below 65%: "This prop type has lower historical accuracy. Recommend monitoring only."
